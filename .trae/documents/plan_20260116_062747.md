## 修复 MCP Client Service 错误

### 问题分析
- 错误发生在 `MCPClientService.ts` 文件的 `getTools()` 方法中
- 错误信息：`Cannot read properties of undefined (reading 'getTools')`
- 根本原因：`webSearchProvider` 变量未初始化，但在 `getTools()` 和 `callTool()` 方法中被调用

### 修复方案
1. **在 MCPClientService 类中添加 webSearchProvider 成员变量**
   - 在第 16-17 行之间添加 `private webSearchProvider: WebSearchProvider;`

2. **在构造函数中初始化 webSearchProvider**
   - 在构造函数中添加 `this.webSearchProvider = new WebSearchProvider();`

3. **导入 WebSearchProvider 类**
   - 在文件顶部添加 `import { WebSearchProvider } from './builtin/WebSearchProvider';`

### 修复步骤
1. 修改 `MCPClientService.ts` 文件，添加必要的导入和初始化代码
2. 保存文件并重新启动开发服务器
3. 验证错误是否已解决

### 预期结果
- 应用能够正常启动，不再出现 `Cannot read properties of undefined (reading 'getTools')` 错误
- MCP 服务能够正常加载工具列表
- 内置的网络搜索工具能够正常使用
