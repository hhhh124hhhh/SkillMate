# 公众号AI助手界面优化方案

## 问题分析

当前界面存在两个主要问题：
1. **功能按钮顺序不合理**：当前顺序为「开始写作 → 寻找选题 → 生成封面 → 文章排版」，不符合正常的创作流程
2. **描述文本不够人性化**：当前文本过于正式，缺乏亲和力

## 优化方案

### 1. 功能按钮重新排序

按照公众号创作的自然流程重新排列按钮顺序：
- **第一步**：寻找选题（热门话题发现）
- **第二步**：定标题（标题生成与优化）
- **第三步**：开始写作（内容创作）
- **第四步**：文章排版（格式美化）

### 2. 新增「定标题」功能按钮

当前界面缺少标题生成功能，需要添加这个重要环节：
- 图标：使用 `Type` 或 `Edit3` 图标
- 颜色：使用紫色系（`text-purple-500`, `bg-purple-50`, `border-purple-100`）
- 操作文本："帮我生成吸引人的标题"

### 3. 优化描述文本

将当前正式的描述文本改为更人性化、亲切的表达：

**原文本**："您的智能公众号创作与运营伙伴。我可以帮您撰写文章、生成封面、管理素材以及更多。"

**优化后**："嗨，我是你的公众号创作小助手！从找选题、想标题到写文章、做排版，我都能陪你一起搞定～"

### 4. 技术实现

修改 `CoworkView.tsx` 文件中的 `EmptyState` 组件：
1. 调整按钮数组的顺序
2. 添加「定标题」按钮对象
3. 更新描述文本
4. 确保新图标正确导入

## 预期效果

- 功能按钮顺序符合创作流程，用户操作更直观
- 新增标题生成功能，完善创作环节
- 描述文本更亲切自然，增强用户好感度
- 整体界面保持原有的设计风格和交互效果
