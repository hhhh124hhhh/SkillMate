# 公众号助手 - 品牌重塑与 UI 升级计划

为了将项目与原有的 "OpenCowork" 区分开，我将进行全面的品牌重塑和 UI 风格升级。目标是打造一个专业、现代且具有科技感的“公众号助手”。

## 1. 视觉风格升级 (Style & Theme)
- **新配色方案**: 
  - **主色 (Primary)**: 从温暖的橙色 (`orange-500`) 切换为专业科技蓝 (`blue-600` / `indigo-600`)。
  - **背景色**: 从暖调的米色 (`#FAF8F5`) 切换为冷调的灰蓝白 (`slate-50` / `#F8FAFC`)。
  - **强调色**: 使用青色 (`teal`) 或紫色 (`violet`) 作为辅助点缀。
- **UI 质感**:
  - 增加圆角 (`rounded-xl` -> `rounded-2xl`)。
  - 使用更轻薄的阴影和更干净的边框。
  - 聊天气泡从“米色”改为“淡蓝色”或“纯白+阴影”。

## 2. Logo 重新设计 (New Logo)
我将创建一个新的 SVG Logo，替换现有的图标。
- **设计概念**: 结合“写作/笔”与“对话/气泡”的元素，或者一个抽象的“助手”形象。
- **文件操作**:
  - 生成 `public/logo_new.svg`。
  - 更新代码引用该 Logo。

## 3. 代码修改清单 (Implementation Tasks)

### 3.1 样式配置 (`tailwind.config.cjs` & `index.css`)
- 修改 CSS 变量中的 `--primary` 和 `--background` 色值。
- 移除硬编码的暖色调类名。

### 3.2 全局布局 (`src/App.tsx`)
- 替换背景色 `bg-[#FAF8F5]` -> `bg-slate-50`。
- 更新 Header 样式，使其更符合新主题。
- 更新 Logo 图片引用。

### 3.3 组件样式 (`src/components/CoworkView.tsx`)
- **聊天气泡**: 修改 `.user-bubble` 样式，使其与新主题融合。
- **按钮**: 将所有 `bg-orange-500` 替换为 `bg-blue-600`。
- **图标颜色**: 更新图标的颜色类名。

### 3.4 交互细节
- 调整“输入框”和“侧边栏”的视觉反馈颜色。

## 4. 预期效果
完成修改后，应用将呈现出一种冷静、高效、专业的工具软件气质，与之前的“温馨协作”风格截然不同。
