## 修复文字显示问题

### 需要修改的内容：

1. **为输入框添加文字颜色**（SettingsView.tsx）
   - 为所有输入框添加 `text-slate-900` 或 `text-black` 类
   - 修改位置：API Key、豆包生图 API Key、API URL、模型名称的输入框

2. **调整标签文字颜色**（SettingsView.tsx）
   - 将 `text-slate-500` 改为 `text-slate-700` 或更深的颜色
   - 修改位置：所有标签文字

3. **调整说明文字颜色**（SettingsView.tsx）
   - 将 `text-slate-400` 改为 `text-slate-600`
   - 修改位置：豆包生图 API Key 和模型名称的说明文字

### 预期效果：
- 输入框文字变为黑色或深灰色，提高可读性
- 标签文字更清晰易读
- 整体界面视觉效果更好
