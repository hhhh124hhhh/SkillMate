**预期结果**:
- ✅ 每次操作都有权限确认

### 场景 3: 多轮对话 可以实现


## ⚠️ 常见问题快速排查

### 问题 1: 应用无法启动
```
错误: SyntaxError: The requested module 'electron' does not provide an export named 'BrowserWindow'
```
**解决方案**: 检查 Node.js 版本
```bash
node --version  # 必须是 v20.x.x
```

### 问题 2: AI 不响应
```
原因: API Key 未配置或无效
```
**解决方案**:
1. 点击设置按钮
2. 配置正确的 API Key
3. 点击保存

### 问题 3: 权限确认弹窗不显示
```
原因: 可能是窗口焦点问题
```
**解决方案**: 点击主窗口激活

### 问题 4: 图片上传失败
```
原因: 文件太大
```
**解决方案**: 压缩图片到 5MB 以下

## 🎯 测试技巧

### 1. 边界测试
- 尝试发送超长文本（1000+ 字符）
- 上传超大图片（10MB+）
- 连续快速发送多条消息

### 2. 错误测试
- 断网后发送消息
- 使用错误的 API Key
- 访问不存在的文件

- 打开任务管理器观察内存占用

- 提示信息是否清晰？
- 操作流程是否直观？
