# 自然文案生成

欢迎使用自然文案生成技能！我可以帮助你去除AI文案的"机械感"，让内容更像人类写的。

## 🎯 核心功能

### 1. 全能去AI味（推荐）

**适用场景**：通用文案优化、公众号文章、自媒体内容

**效果**：组合所有去AI味技巧，全面优化文案

**使用方式**：
```
请使用"全能去AI味"模式优化这篇文章：
[粘贴你的文案]
```

**特点**：
- ✅ 避免AI套路化表达（首先、其次、综上所述）

## 📋 去AI味核心技巧

### DO（推荐做法）

✅ **使用高多样性和高节奏感**
- 词汇丰富多变
- 长短句交替
- 避免固定模式

✅ **明确情感基调和场景设定**
- 温暖、感人、激励、幽默
- 在开头就设定基调
- 具体场景而非抽象描述

✅ **增加细节描写和人物对话**
- 视觉、听觉、嗅觉
- 人物动作和表情
- 真实对话特征（打断、犹豫、笑场）

✅ **使用比喻和修辞手法**
- 用自然现象隐喻（破茧、潮汐）
- 避免直接说出情绪名称
- 让读者自己感受

✅ **口语化表达，加入情感词汇**
- "说实话"、"emm"、"啊对了"
- "这让我很震撼"
- 个人观点和立场

✅ **多用短句，删除总结**
- 每句一个意思
- 避免长难句
- 自然收尾，可以不总结

### DON'T（避免做法）

❌ **使用AI高频词汇**
- "首先、其次、最后"
- "综上所述、总而言之"
- "值得注意的是"、"显而易见"

❌ **使用总结性词汇**
- "总而言之"
- "综上所述"
- "总得来说"

❌ **过度使用专业术语**
- "矩阵"、"输出"、"生态链路"
- 除非必要，否则用通俗表达

❌ **使用长难句和复杂结构**
- 复合句、从句堆叠
- 超过3行的句子
- 逻辑嵌套

❌ **缺乏具体例子和数据支撑**
- 空洞说教
- 没有具体案例
- 缺乏数据

❌ **过度完美的语法结构**
- 太工整
- 没有人类说话的特点
- 缺乏口语化表达

## 🎨 实用技巧示例

### 技巧1：打造"人设"

**公式**：角色 + 内容 + 态度 + 语气 + 情绪

**示例**：
```
你是一名资深产品经理，对于AI工具替代设计师持怀疑态度。
请就AI设计工具的发展分享观点，
语气像是在和产品团队交流，
整体语调必须展现出担忧但理性的情绪。
```

### 技巧2：增加细节描写

**差的写法**：
```
那是一个秋天的傍晚，风景很美。
```

**好的写法**：
```
那是一个秋天的傍晚，天空呈现出渐变的橙色和紫色。
树叶已经变成了金黄色，随着微风轻轻飘落。
空气中弥漫着桂花香和烤红薯的味道。
散步的孩子们踩着落叶，发出沙沙的响声。
```

### 技巧3：使用对话增强真实感

**差的写法**：
```
他们多年后重逢，回忆起了高中生活。
```

**好的写法**：
```
"哎，你还记得高三那个食堂吗？"小明突然笑了。
"怎么可能忘！那盘子里的菜...算了不说了，"我笑着摇摇头。
"哈哈，还有那个数学老师，永远擦不干净的黑板..."
"停停停，你别说了，我PTSD都要犯了！"
我们两个人在咖啡馆里笑成一团，周围的人都投来好奇的目光。
```

## 📊 使用建议

### 何时使用本技能？

**推荐场景**：
- ✅ 公众号文案创作
- ✅ 社交媒体内容
- ✅ 个人分享和故事
- ✅ 情感激发类内容
- ✅ 已生成内容的后期润色

**不推荐场景**：
- ❌ 严格的学术论文（需要正式性）
- ❌ 法律文档（需要准确性）
- ❌ 技术文档（需要清晰性）

### 最佳实践

1. **第一次生成时就可以使用**
   - 在提示词中就说明去AI味要求
   - 比后期修改效果更好

2. **结合联网搜索**
   - 先搜索最新数据和案例
   - 再用自然风格生成
   - 内容会更真实

3. **保持品牌调性**
   - 去AI味不代表失去专业度
   - 根据目标读者调整语气

4. **人工审校仍是必要**
   - AI只能辅助，不能完全替代人工
   - 检查事实准确性
   - 确保不改变核心信息
