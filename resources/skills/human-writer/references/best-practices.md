## 一、DO & DON'T 清单

### ✅ DO（推荐做法）

#### 1. 使用高多样性和高节奏感

**为什么**：AI基于概率生成文本，倾向于输出"平均值"内容，导致文案单调、缺乏变化。

**怎么做**：
- 词汇丰富多变，避免重复
- 长短句交替，创造节奏感
- 变换句子结构（陈述、疑问、感叹、倒装）
- 使用同义词替换

**示例**：
```
❌ 差的写法：
AI工具很好用。AI工具可以提高效率。AI工具可以节省时间。

✅ 好的写法：
说实话，AI工具确实好用。不仅效率提升明显，还能节省大量时间。
最让我惊喜的是，它让繁琐的工作变得简单了。
```

---

#### 2. 明确情感基调和场景设定

**为什么**：AI生成的内容往往缺乏情感，像说明文而不是有温度的文字。

**怎么做**：
- 在开头就设定情感基调（温暖、感人、激励、幽默）
- 描述具体场景而非抽象概念
- 加入个人感受和立场

**示例**：
```
❌ 差的写法：
本产品具有良好的性能和用户体验。

✅ 好的写法：
第一次用这个产品时，我有点小震惊。
居然有人能把产品设计得这么贴心！
每个细节都像是为我量身定制的，这种感觉真的很难得。
```

---

#### 3. 增加细节描写和人物对话

**为什么**：细节让内容更真实，对话让场景更生动。

**怎么做**：
- 描述视觉、听觉、嗅觉
- 人物动作和表情
- 使用真实对话特征（打断、犹豫、笑场）

**示例**：
```
❌ 差的写法：
他们多年后重逢，回忆起了高中生活。

✅ 好的写法：
"哎，你还记得高三那个食堂吗？"小明突然笑了。
"怎么可能忘！那盘子里的菜...算了不说了，"我笑着摇摇头。
"哈哈，还有那个数学老师，永远擦不干净的黑板..."
```

---

#### 4. 使用比喻和修辞手法

**为什么**：比喻让抽象概念变得具体，修辞让内容更生动。

**怎么做**：
- 用自然现象隐喻（破茧、潮汐、四季更替）
- 避免直接说出情绪名称
- 让读者自己感受

**示例**：
```
❌ 差的写法：
他感到很悲伤。

✅ 好的写法：
那天下午，他坐在窗边，看着窗外的雨。
雨水划过玻璃，像一道道泪痕。
那一刻，他突然明白，有些失去，就像秋天的落叶，
没有声音，却无处不在。
```

---

#### 5. 口语化表达，加入情感词汇

**为什么**：人类说话是自然的，不会像书面语那样工整。

**怎么做**：
- 加入口语化词汇（"说实话"、"emm"、"啊对了"）
- 加入个人观点和情感表达
- 使用情感词汇（"这让我很震撼"）

**示例**：
```
❌ 差的写法：
综上所述，该产品具有良好的市场前景。

✅ 好的写法：
说实话，这产品真的有点东西。
我原本以为只是又一个噱头，
结果用了之后发现，它确实解决了很多实际问题。
```

---

#### 6. 多用短句，删除总结

**为什么**：人类说话不会那么工整，也不会每段都总结。

**怎么做**：
- 多用短句，每句一个意思
- 删除"综上所述"、"总而言之"
- 自然收尾，可以不总结

**示例**：
```
❌ 差的写法：
首先，AI可以提高效率。其次，AI可以减少错误。最后，AI可以降低成本。
综上所述，AI技术在企业应用中具有重要价值。

✅ 好的写法：
AI确实能提高效率。这点大家应该都有体会。
更关键的是，它还能减少很多低级错误。
成本方面，虽然前期投入不低，但长期来看是值得的。
```

---

### ❌ DON'T（避免做法）

#### 1. 不要使用"首先、其次、最后"等逻辑词

**为什么**：这些词是AI的标志性特征，人类很少这样说话。

**示例**：
```
❌ 避免：
首先，我们需要关注用户体验。
其次，要提升产品性能。
最后，优化成本结构。

✅ 替代：
用户体验是核心。
产品性能也得跟上。
成本当然也不能忽视。
```

---

#### 2. 不要使用"综上所述"等总结性词汇

**为什么**：人类写文章不一定会总结，尤其是不用这些模板化词汇。

**示例**：
```
❌ 避免：
综上所述，该产品具有良好的市场前景。

✅ 替代：
（自然收尾，不总结）
或者：
这款产品，值得期待。
```

---

#### 3. 不要过度使用专业术语

**为什么**：专业术语会让内容显得生硬，不够自然。

**示例**：
```
❌ 避免：
该产品通过矩阵化输出，优化了生态链路。

✅ 替代：
这个产品让各个环节配合得更顺畅。
```

---

#### 4. 不要使用长难句和复杂结构

**为什么**：人类说话不会那么复杂，长难句是书面语的特征。

**示例**：
```
❌ 避免：
虽然该产品在初期投入方面存在一定挑战，
但考虑到其长期收益和市场份额的潜在增长，
我们认为这是一个值得投资的项目。

✅ 替代：
这产品前期投入确实不小。
但长远来看，收益是可观的。
市场份额也有很大提升空间。
所以，值得投。
```

---

#### 5. 不要缺乏具体例子和数据支撑

**为什么**：空洞的论述没有说服力，像AI生成的。

**示例**：
```
❌ 避免：
该产品具有良好的性能。

✅ 替代：
该产品响应速度提升了50%，用户满意度达到92%。
```

---

#### 6. 不要过度完美的语法结构

**为什么**：人类说话有停顿、有重复、不完美，这才有真实感。

**示例**：
```
❌ 避免：
该产品的所有功能均已实现，且运行稳定。

✅ 替代：
功能基本都实现了。运行还算稳定吧，
偶尔小卡一下，但问题不大。
```

## 二、场景化最佳实践

### 场景1：公众号文章

**目标**：让文章像真人写的，有观点、有情感、有细节

**DO**：
- ✅ 开头用热点或个人故事引入
- ✅ 加入个人观点和立场
- ✅ 使用口语化表达
- ✅ 增加具体案例和数据
- ✅ 结尾自然，可以不总结

**DON'T**：
- ❌ 不要"首先、其次、最后"
- ❌ 不要"综上所述"
- ❌ 不要空洛建议
- ❌ 不要过度完美

**示例**：
```
✅ 好的开头：
昨天看到个新闻，挺有意思的。
智谱上市了，市值579亿港元。
这消息让我想起了3个月前在黑客松现场...

✅ 好的结尾：
就这样吧。
下次再聊。
```

---

### 场景2：社交媒体内容

**目标**：让内容像朋友聊天，轻松、自然、有互动

**DO**：
- ✅ 开头用"你知道吗"、"说实话"
- ✅ 加入情感词汇和表情符号
- ✅ 多用短句和反问句
- ✅ 鼓励互动和评论

**DON'T**：
- ❌ 不要正式用语
- ❌ 不要长段落
- ❌ 不要逻辑词
- ❌ 不要总结

**示例**：
```
✅ 好的写法：
说实话，这事儿我还真没想明白。
你们觉得呢？
评论区聊聊呗～
```

---

### 场景3：技术报告

**目标**：保持专业性的同时提升可读性

**DO**：
- ✅ 简化句子结构
- ✅ 减少专业术语
- ✅ 增加具体数据
- ✅ 保持逻辑清晰

**DON'T**：
- ❌ 不要过度口语化
- ❌ 不要失去专业性
- ❌ 不要改变核心数据
- ❌ 不要模糊逻辑

**示例**：
```
✅ 好的写法：
本报告通过3种实验方法，验证了AI模型的有效性。
准确率达到95%，响应时间减少60%。
数据来源：2025年Q1测试结果。
```

## 三、分步骤实施指南

### Step 1：分析原文

**检查AI味**：
- [ ] 是否有"首先、其次、最后"？
- [ ] 是否有"综上所述"？
- [ ] 是否有专业术语堆砌？
- [ ] 是否有长难句？
- [ ] 是否缺乏具体例子？

**标注问题**：
用笔或高亮标记出AI味重的地方

---

### Step 2：选择模板

根据场景选择合适的模板：
- 公众号文章 → 模板1：全能去AI味
- 情感激发 → 模板2：情感文案专用
- 后期润色 → 模板3：快速润色
- 技术报告 → 模板4：学术降重
- 社交媒体 → 模板5：口语化改写

---

### Step 3：生成优化版

使用选定的模板，让AI重新生成

---

### Step 4：人工审校

**检查点**：
- [ ] 是否保留了核心信息？
- [ ] 是否更自然了？
- [ ] 是否有情感共鸣？
- [ ] 事实是否准确？
- [ ] 是否过度口语化？

**微调**：
根据检查结果进行人工调整

---

### Step 5：最终验证

**AI检测率**：使用检测工具（可选）
**读者反馈**：让他人阅读并反馈
**对比效果**：对比优化前后的差异

## 四、常见错误及纠正

### 错误1：过度口语化

**问题**：为了去AI味，把内容改得太随意，失去专业性

**纠正**：
- 保持品牌调性
- 根据目标读者调整语气
- 去AI味≠失去专业度

**示例**：
```
❌ 过度口语化：
这产品超牛逼，简直炸裂！

✅ 适度口语化：
这产品确实有点东西。
性能提升明显，体验也很好。
```

---

### 错误2：改变核心信息

**问题**：润色时改变了原文的核心信息和数据

**纠正**：
- 先提取核心信息和数据
- 润色后核对是否保留
- 必要时人工修正

---

### 错误3：为了去AI味而去除逻辑

**问题**：删除了所有逻辑连接，导致文章混乱

**纠正**：
- 保留必要的逻辑关系
- 用自然的过渡方式
- 确保文章结构清晰

---

### 错误4：过度使用情感词汇

**问题**：每句话都加"说实话"、"emm"，显得刻意

**纠正**：
- 适度使用情感词汇
- 在关键位置使用
- 不要堆砌

## 六、进阶技巧

### 技巧1：学习优秀范文

**方法**：
1. 收集10篇你认为"自然"的文章
2. 分析其写作特点
3. 提取共性技巧
4. 应用到自己的写作中

---

### 技巧2：建立个人风格库

**方法**：
1. 记录自己常用的表达方式
2. 形成个人化的词汇库
3. 在AI生成时要求模仿这种风格

---

### 技巧3：多轮优化

**方法**：
1. 第一次生成：使用全能去AI味模板
2. 如果还有AI味：使用快速润色模板
3. 人工审校：微调和最终优化
