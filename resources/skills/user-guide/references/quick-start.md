# 5分钟快速入门

> 欢迎使用公众号创作系统！本指南将帮助你在5分钟内快速上手，完成你的第一篇文章。

---

## 目标

通过本指南，你将学会：
1. ✅ 创建一个公众号文章项目
2. ✅ 生成吸引人的标题
3. ✅ 撰写完整的文章内容
4. ✅ 优化文章排版

**预计时间**：5分钟

---

## 第一步：创建项目（30秒）

使用以下命令创建你的第一个项目：

```
创建公众号项目 "我的第一篇文章"
```

**系统会做什么**：
- 创建项目文件夹 `公众号项目/我的第一篇文章/`
- 生成4个标准文件：
  - `00_背景资料.md` - 存放参考资料
  - `01_文章大纲.md` - 文章结构大纲
  - `02_文章内容.md` - 完整文章内容
  - `03_发布记录.md` - 发布数据和反馈
- 创建 `assets/` 目录存放图片和素材

**验证**：检查项目文件夹是否创建成功

---

## 第二步：选择选题（1分钟）

使用选题工具推荐热点：

```
推荐当前热门选题
```

或者自己指定主题：

```
收集背景资料 "AI写作技巧"
```

**系统会做什么**：
- 搜索相关热点和资料
- 整理关键数据和案例
- 保存到 `00_背景资料.md`

**验证**：打开背景资料文件，查看收集的内容

---

## 第三步：生成标题（1分钟）

使用标题生成器创建吸引人的标题：

```
生成标题 "AI写作技巧" 10个
```

**系统会做什么**：
- 生成10个候选标题
- 每个标题都有点击率评分
- 按热度排序展示

**选择建议**：
- 选择评分前3的标题
- 或者使用多轮生成：`再生成10个标题`

**验证**：查看生成的标题列表

---

## 第四步：撰写内容（2分钟）

使用AI写作助手生成文章：

```
帮我写一篇关于AI写作的文章
```

**或者更具体的指令**：

```
写一篇文章，主题是AI写作技巧，包含以下要点：
1. AI写作的优势
2. 如何使用AI提高写作效率
3. AI写作的注意事项

字数：2000字左右
风格：专业但不生硬
```

**系统会做什么**：
- 基于你的要求生成完整文章
- 自动分段和排版
- 保存到 `02_文章内容.md`

**验证**：打开文章内容文件，检查质量

---

## 第五步：优化排版（30秒）

使用智能排版优化文章：

```
优化排版
```

**系统会做什么**：
- 应用2026年最新排版规范
- 调整段落间距
- 添加合适的留白
- 优化标题层级

**验证**：查看文章排版效果

---

## 恭喜！🎉

你已经完成了第一篇文章的创作！

### 下一步建议

1. **查看文章**：打开 `公众号项目/我的第一篇文章/02_文章内容.md`

## 常见问题

### Q1：项目文件在哪里？
**A**：默认在 `公众号项目/` 目录，相对于项目根目录。

### Q2：如何修改已生成的内容？
**A**：直接编辑 `02_文章内容.md` 文件，或使用AI润色命令。

### Q3：可以生成不同风格的标题吗？
**A**：可以！使用 `生成标题 "主题" 10个，风格="震惊体"`

### Q4：文章质量不满意怎么办？
**A**：尝试以下方法：
1. 使用更具体的指令
2. 分步骤生成（先大纲再内容）
3. 使用 `帮我润色这篇文章` 优化

### Q5：如何自定义路径？
**A**：编辑 `.claude/config.json` 文件，修改路径配置。

## 故障排查

### 问题1：命令没有反应

**可能原因**：
- Skill未正确加载
- 命令格式错误

**解决方法**：
1. 检查命令格式是否正确
2. 查看错误提示信息
3. 重启Claude Code

### 问题2：项目文件夹未创建

**可能原因**：
- 路径不存在
- 权限不足

**解决方法**：
1. 检查 `公众号项目/` 目录是否存在
2. 确保有写入权限
3. 手动创建目录

### 问题3：生成的内容质量差

**可能原因**：
- 指令不够具体
- 背景资料不足

**解决方法**：
1. 提供更详细的指令
2. 先收集背景资料
3. 使用风格学习功能
