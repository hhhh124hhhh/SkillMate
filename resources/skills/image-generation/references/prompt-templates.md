### 微信公众号封面

> **⚠️ 重要提示**：微信公众号封面的特殊要求
> - **比例**：2.35:1 横版（推荐 900x383 或 1080x460）
> - **双重场景**：同一张图用于头条展示 + 分享卡片
> - **自动裁剪**：分享时会自动裁剪为正方形，四周可能被切掉
> - **设计原则**：关键内容必须放在中心区域

**模板**：

```
微信公众号封面图，主题：{文章主题}
横版布局，2.35:1比例
尺寸：900像素宽，383像素高

【重要】设计原则：
- 所有关键元素放在画面中心区域
- 文字和图标距离边缘至少15%的安全距离
- 确保裁剪为正方形后核心内容依然完整
- 避免在左右两侧放置重要文字

中心区域（核心内容）：
{主要视觉元素}
- 位于画面正中央
- 占据画面中心60%的区域
- 包含最重要的标题和图标

中心元素设计：
- 主标题：{文章核心标题}
  使用大字号，醒目清晰
  位于画面正中心
  确保裁剪后依然完整

- 副标题：{补充说明或核心数据}
  位于主标题下方或旁边
  字号略小于主标题
  简短精炼

- 核心图标：{主题相关图标}
  位于中心区域
  与文字形成平衡构图

背景：
从{颜色1}到{颜色2}的渐变
营造{科技感/专业感}氛围
简洁不抢眼

边缘区域（留白或装饰）：
左右两侧可以添加装饰元素：
{星星、光点、几何图形}
但不能是关键内容

整体风格：
扁平化设计，现代科技风
清晰易读，有视觉冲击力
关键内容集中，边缘简洁
```

**示例1：智谱上市主题**

```
微信公众号封面图，主题：智谱上市与GLM-4.7
横版布局，2.35:1比例
尺寸：900x383

【重要】设计原则：
所有关键元素放在画面中心，确保分享裁剪后依然完整

中心区域设计：
- 主标题：智谱上市579亿
  字号最大，位于正中央
  使用白色或亮色

- 副标题：GLM-4.7实测
  位于主标题下方
  字号较小

- 核心图标：
  左侧：智谱AI图标（简化版）
  右侧：代码符号</>或AI芯片
  夹住标题文字

背景：
从深蓝色(#1E3A8A)到紫色(#7C3AED)的渐变
营造科技感

边缘装饰：
左右两侧：淡蓝色的几何图形或光点
不影响中心区域的识别

整体风格：
科技感，蓝色系
关键内容集中中央
边缘简洁留白
```

**示例2：AI编程技巧主题**

```
微信公众号封面图，主题：AI编程技巧
横版布局，2.35:1比例
尺寸：900x383

中心区域：
主标题：6个AI编程技巧
副标题：从DeepSeek到GLM
图标：代码编辑器图标（中心）

背景：
从绿色到青色的渐变
营造成长感

边缘：
左右两侧：淡色的代码符号装饰
```

**设计技巧**：

1. **网格辅助设计**：
   - 画面中心60%区域为安全区
   - 四周各20%为可能被裁剪的区域
   - 关键内容只放在安全区内

2. **文字排版**：
   - 主标题居中，字号最大
   - 副标题在主标题下方
   - 避免文字延伸到画面边缘

3. **颜色对比**：
   - 使用高对比度颜色
   - 确保在不同背景下都清晰
   - 避免渐变覆盖重要文字

4. **图标位置**：
   - 主要图标放在中心
   - 装饰性图标可以放在边缘
   - 保持整体平衡

**常见错误**：

❌ **错误1**：文字延伸到边缘
```
标题：智谱上市579亿！我凭GLM斩获黑客松奖
（文字太长，分享后可能被截断）
```

✅ **正确**：精炼标题居中
```
标题：智谱上市579亿
副标题：GLM实测
（短小精悍，居中显示）
```

❌ **错误2**：关键元素在两侧
```
左侧：智谱图标  中间：GLM  右侧：获奖证书
（两侧图标可能被裁掉）
```

✅ **正确**：所有关键元素居中
```
中心：智谱上市 + GLM图标
（集中中央，裁剪无忧）
```

❌ **错误3**：使用1:1正方形
```
生成1024x1024的图片
（不符合微信要求的2.35:1）
```

✅ **正确**：使用2.35:1横版
```
生成900x383的图片
（符合微信要求，兼顾分享）
```

**API调用示例**：

```python
# 生成微信公众号封面

    size="900x383",  # 2.35:1 比例
    quality="hd"
)
```

**测试建议**：

生成后检查：
1. ✅ 关键文字是否在中心区域
2. ✅ 是否可以安全地裁剪为正方形
3. ✅ 小尺寸下文字是否依然清晰
4. ✅ 颜色对比是否足够

## 💡 使用建议

### 模板替换规则

1. **占位符替换**：
   - `{系统名称}` → 具体的系统名称
   - `{层数}` → 具体的数字
   - `{技术A}` → 第一个技术/方案的名称
   - `{技术B}` → 第二个技术/方案的名称

2. **细节补充**：
   - 在使用模板时，根据实际情况补充更多细节
   - 细节越多，生成结果越精确

3. **风格调整**：
   - 根据品牌或个人喜好调整配色方案
   - 保持整体风格一致性

### Prompt优化技巧

1. **从简单开始**：先用基础模板生成，看效果后再优化
2. **逐步添加细节**：每次只添加1-2个细节，观察变化
3. **参考优秀案例**：学习prompt模板中的示例
4. **保持一致性**：同一系列配图使用相似风格
