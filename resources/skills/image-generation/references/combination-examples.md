## 示例1: 专业科技 - 技术文章封面

**使用场景**：技术文章、AI 主题、编程教程

**主题**：智谱上市与 GLM-4.7 实测

### 完整提示词

```
微信公众号封面图，主题：智谱上市与GLM-4.7实测
横版布局，2.35:1比例
尺寸：900×383

【小红书风格要求】
- 文字清晰可读，重要信息突出显示
- 排版美观，留白合理
- 高对比度配色，确保可读性
- 标题大而醒目

【公众号特殊要求】
- 所有关键元素放在画面中心区域
- 文字和图标距离边缘至少15%的安全距离
- 确保裁剪为正方形后核心内容依然完整
- 避免在左右两侧放置重要文字

【专业科技风格样式】
背景：从深蓝色(#1E3A8A)到紫色(#7C3AED)的水平渐变
文字颜色：白色(#FFFFFF)
装饰元素：
  - 左侧：电路板纹路、代码符号 </> { }
  - 右侧：AI芯片图标、光点
  - 透明度：30%
字体：现代无衬线，科技感

【中心内容布局】
主标题：智谱上市579亿
  - 字号最大，位于正中央
  - 白色文字，醒目清晰

副标题：GLM-4.7实测
  - 位于主标题下方
  - 字号较小，浅蓝白色(#E0E7FF)

核心图标：
  - AI芯片图标（中心）
  - 与文字形成平衡构图
  - 简洁的芯片设计

【整体风格】
- 扁平化设计，现代科技风
- 清晰易读，有视觉冲击力
- 关键内容集中中央
- 边缘科技感装饰，可被裁剪

【技术规格】
- 比例：2.35:1 (900×383)
- 高清画质
- 确保裁剪为1:1后标题完整
```

### 使用建议

- ✅ 适合技术教程、AI 主题文章
- ✅ 强调数据和指标时使用
- ⚠️ 确保数据准确无误
- ⚠️ 避免过多技术术语

### 预期效果

- 深蓝到紫色渐变背景，科技感强
- 白色文字清晰醒目
- AI 芯片图标增强主题表达
- 电路板、代码符号装饰强化技术氛围
- 裁剪为正方形后标题依然完整

### 常见问题

**Q: 可以混合不同风格吗？**

A: 不建议。同一篇文章的封面和配图应该保持风格一致，以建立统一的视觉识别。

**Q: 可以修改配色方案吗？**

A: 可以，但要注意：
- 保持对比度 ≥ 4.5:1
- 保持风格的整体性
- 不要偏离原有风格太远

**Q: 如何添加更多内容？**

A: 建议：
- 控制信息密度，不要过度拥挤
- 使用多张配图而不是一张
- 保持足够的留白

## 提示词优化技巧

### 1. 提高生成质量

**具体化描述**：
```
❌ 模糊："添加一些装饰元素"
✅ 具体："左侧添加电路板纹路，右侧添加AI芯片图标，透明度30%"
```

**明确的布局**：
```
❌ 模糊："标题放在中间"
✅ 具体："主标题位于正中央，字号最大；副标题在主标题下方，字号为标题的60%"
```

### 2. 确保风格一致性

**使用模板**：
- 从本示例文档选择最接近的模板
- 只替换内容，不改变风格
- 保持配色、字体、装饰元素一致

**系列文章**：
- 同一系列文章使用相同风格
- 保持配色方案统一
- 使用相似的装饰元素

### 3. 优化视觉效果

**对比度检查**：
- 文字与背景对比度 ≥ 7:1（推荐）
- 最低 ≥ 4.5:1（AA级）
- 使用在线工具验证

**留白控制**：
- 文字周围至少保留 10-15% 的留白
- 标题与正文之间有明显的间距
- 整体布局不拥挤
