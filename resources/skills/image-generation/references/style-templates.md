### 基本信息定位

| 属性 | 值 |
|------|-----|
| 风格代码 | `tech` |
| 风格名称 | 专业科技 |
| 适合场景 | 技术文章、AI主题、编程教程、架构分析、数据报告 |
| 目标受众 | 开发者、技术从业者、科技爱好者 |
| 情感基调 | 专业、前沿、创新、可信 |

### 配色方案

```yaml
主背景渐变:
  起点: "#1E3A8A"  # 深蓝色
  终点: "#7C3AED"  # 紫色
  方向: "水平渐变"

  主文字: "#FFFFFF"     # 白色
  副文字: "#E0E7FF"     # 浅蓝白色
  强调色: "#60A5FA"     # 亮蓝色

```

**配色心理学**：
- 深蓝色：专业、可信赖、科技感
- 紫色：创新、高端、神秘
- 白色：清晰、现代、科技
- 亮蓝：活力、未来、能量

### 字体风格

```yaml
字体类型: "现代无衬线"
字重: "Medium - Bold"
字间距: "正常"
字母间距: "略宽松"

特点:
  - 现代感强
  - 科技感明显
  - 清晰易读
  - 适合屏幕显示
```

### 装饰元素库

```yaml
主要元素:
  - 几何图形:
    - 六边形网格
    - 电路板纹路
    - 技术线条

  - 图标符号:
    - 代码符号: </> { } [ ]
    - AI符号: 大脑、芯片、网络节点
    - 数据符号: 图表、曲线、仪表盘

    - 光点: 小圆点，低透明度
```

## 风格自动匹配算法

### 关键词映射

```python
STYLE_KEYWORDS = {
    "tech": [
        "技术", "AI", "代码", "编程", "架构", "算法",
        "数据", "开发", "系统", "平台", "工具", "GLM",
        "GPT", "深度学习", "机器学习", "API"
    ],
    "fresh": [
        "生活", "日常", "分享", "成长", "学习", "笔记",
        "技巧", "经验", "入门", "轻科普", "新手"
    ],
    "minimal": [
        "极简", "哲学", "思考", "观点", "理论", "抽象",
        "本质", "深度", "探讨", "分析"
    ],
    "warm": [
        "情感", "感悟", "心情", "治愈", "温暖", "故事",
        "陪伴", "成长记录", "人生", "随笔"
    ],
    "business": [
        "商业", "市场", "数据报告", "分析", "行业", "趋势",
        "投资", "财报", "解读", "研究", "洞察"
    ]
}
```

### 匹配逻辑

```python
def auto_match_style(content: str) -> str:
    """基于内容自动匹配风格"""
    scores = {}

    for style, keywords in STYLE_KEYWORDS.items():
        score = sum(1 for kw in keywords if kw in content)
        scores[style] = score

    # 返回得分最高的风格
    max_score = max(scores.values())
    if max_score > 0:
        return max(scores, key=scores.get)
    else:
        return "business"  # 默认商务风格
```
