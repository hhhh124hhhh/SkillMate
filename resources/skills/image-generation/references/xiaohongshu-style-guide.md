# 小红书风格设计规范

本文档从小红书风格的提示词中提取核心设计原则，用于指导公众号封面和配图的设计。

---

## 核心设计理念

小红书风格强调：
- **清新精致**：视觉设计感强，适合年轻人审美
- **清晰易读**：文字层级分明，信息传达准确
- **视觉吸引力**：配色和谐，有设计感但不花哨
- **专业感与亲和力并存**：既保持专业度，又不过于严肃

---

## 一、文字排版要求

### 1.1 文字层级

**原则**：清晰的视觉层级，让重要信息一目了然

```
层级1：主标题
- 字号最大
- 位于视觉中心
- 使用最醒目的颜色

层级2：副标题/核心数据
- 字号中等（主标题的60-70%）
- 紧邻主标题下方或旁边
- 辅助说明或强调重点

层级3：正文/列表
- 字号最小（主标题的40-50%）
- 用于详细说明或列表项
- 保持良好可读性
```

**应用示例**：
```
主标题：智谱上市579亿
副标题：GLM-4.7实测
列表项：
  • Code Arena 第1名
  • 超过GPT-5.2
```

### 1.2 文字可读性

**关键要求**：
- ✅ 文字清晰可读，字号适中
- ✅ 重要信息突出显示（颜色、字号、位置）
- ✅ 排版美观，留白合理
- ✅ 支持 emoji 和符号（用于增强表达）
- ❌ 避免文字过小或过于拥挤
- ❌ 避免使用过多颜色导致视觉混乱

**可读性测试**：
- 在小尺寸下（如手机屏幕）文字依然清晰
- 主标题一眼可识别
- 次要信息不会喧宾夺主

### 1.3 留白原则

**核心思想**：留白不是浪费，而是设计的一部分

**建议**：
- 文字周围至少保留 10-15% 的留白
- 标题与正文之间有明显的间距
- 列表项之间有清晰的分隔
- 整体布局不拥挤，有呼吸感

**示例**：
```
┌─────────────────────────────────┐
│                                 │
│         智谱上市579亿           │  ← 顶部留白
│                                 │
│       GLM-4.7实测              │  ← 层级间距
│                                 │
│   • Code Arena 第1名           │  ← 列表间距
│   • 超过GPT-5.2               │
│                                 │
└─────────────────────────────────┘
```

## 二、视觉元素设计

### 2.1 背景设计

**原则**：简洁但不单调

**推荐方案**：
1. **渐变背景**（首选）
   - 从颜色A到颜色B的柔和渐变
   - 渐变方向：水平、垂直、对角线
   - 避免过于鲜艳或对比过强的渐变

2. **纯色背景**
   - 使用低饱和度的颜色
   - 避免纯黑或纯白
   - 可以添加细微的纹理

3. **背景装饰**
   - 可以添加小图标、几何图形
   - 装饰元素不能干扰主要内容
   - 保持透明度，不要过于突出

**配色参考**：
- 清新风：薄荷绿 → 暖黄
- 科技风：深蓝 → 紫色
- 温暖风：暖黄 → 粉红

### 2.2 装饰元素

**目的**：增强视觉吸引力，不喧宾夺主

**常用元素**：
1. **几何图形**
   - 圆形、方形、三角形
   - 线条、网格、波浪线
   - 作为背景或边框装饰

2. **图标插画**
   - 与主题相关的简化图标
   - 扁平化风格，保持一致
   - 不宜过多，2-3个即可

3. **emoji 和符号**
   - ✨、💡、🚀、📊 等
   - 用于列表项或强调重点
   - 适量使用，不要滥用

4. **光效和质感**
   - 光点、光晕、渐变光
   - 柔和的阴影
   - 轻微的质感（如纸张、磨砂）

**使用原则**：
- 装饰元素必须服务于内容，不能分散注意力
- 保持风格统一性
- 不影响文字可读性

### 2.3 配色方案

**配色原则**：
1. **高对比度**：确保文字清晰可读（对比度 ≥ 4.5:1）
2. **和谐统一**：使用相邻色或同类色
3. **情感表达**：通过配色传达内容氛围
4. **品牌一致**：保持系列文章的配色统一

**推荐配色组合**：

#### 清新活泼
```
背景：薄荷绿(#A7F3D0) → 暖黄(#FCD34D)
文字：深绿(#065F46)
强调：橙色(#F59E0B)
```

#### 专业科技
```
背景：深蓝(#1E3A8A) → 紫色(#7C3AED)
文字：白色(#FFFFFF)
强调：亮蓝(#60A5FA)
```

#### 温暖治愈
```
背景：暖黄(#FDE68A) → 粉红(#FCA5A5)
文字：深棕(#92400E)
强调：橙色(#F59E0B)
```

#### 简约极简
```
背景：浅灰白(#F9FAFB)
文字：深灰(#1F2937)
强调：蓝色(#3B82F6) - 少量使用
```

#### 商务专业
```
背景：深蓝(#1E40AF) 或 深灰(#1F2937)
文字：白色(#FFFFFF) 或 浅灰(#F9FAFB)
强调：蓝色(#3B82F6)
```

## 三、页面类型差异化

### 3.1 封面页 (Cover)

**设计重点**：吸引注意力，传达核心信息

**特点**：
- 标题占据主要位置，字号最大
- 副标题居中或在标题下方
- 整体设计要有吸引力和冲击力
- 背景可以更丰富，有视觉焦点
- 支持较大的装饰元素

**布局示例**：
```
┌─────────────────────────────────────┐
│                                     │
│         智谱上市579亿              │  ← 主标题，最大字号
│       GLM-4.7实测                 │  ← 副标题，中等字号
│                                     │
│        [AI芯片图标]                │  ← 核心视觉元素
│                                     │
└─────────────────────────────────────┘
```

### 3.2 内容页 (Content)

**设计重点**：信息层次清晰，易于阅读理解

**特点**：
- 信息层次分明
- 列表项清晰展示
- 重点内容用颜色或粗体强调
- 可以有小图标辅助说明
- 布局结构化，便于扫描

**布局示例**：
```
┌─────────────────────────────────────┐
│     AI编程的6个技巧                │  ← 标题
│                                     │
│  1. 🚀 使用结构化提示词           │  ← 列表项
│  2. 💡 分解复杂任务               │
│  3. 🔄 迭代优化                   │
│  4. 📝 添加示例代码               │
│  5. ⚡ 优化响应速度               │
│  6. 🎯 测试验证结果               │
│                                     │
└─────────────────────────────────────┘
```

### 3.3 总结页 (Summary)

**设计重点**：给人完成感和成就感

**特点**：
- 总结性文字突出
- 可以有勾选框或完成标志（✓）
- 鼓励性的视觉元素
- 正面积极的氛围
- 强调收获和成长

**布局示例**：
```
┌─────────────────────────────────────┐
│                                     │
│        ✓ 学习完成！                 │  ← 完成标志
│                                     │
│      你已经掌握了：                 │
│                                     │
│      ✓ AI编程基础                  │
│      ✓ 6个实用技巧                 │
│      ✓ 最佳实践                     │
│                                     │
│      🎉 继续加油！                  │  ← 鼓励性元素
│                                     │
└─────────────────────────────────────┘
```

## 四、对比度原则

### 4.1 可读性要求

**WCAG 标准**：
- AAA级：对比度 ≥ 7:1（推荐）
- AA级：对比度 ≥ 4.5:1（最低要求）

**测试方法**：
- 使用在线对比度检查器
- 在不同设备上测试
- 考虑色盲用户

### 4.2 常见配色组合对比度

| 背景色 | 文字色 | 对比度 | 可用性 |
|-------|--------|--------|--------|
| #FFFFFF | #1F2937 | 16.4:1 | ✅ 优秀 |
| #1E3A8A | #FFFFFF | 11.2:1 | ✅ 优秀 |
| #A7F3D0 | #065F46 | 8.7:1 | ✅ 优秀 |
| #FDE68A | #92400E | 6.8:1 | ✅ 良好 |
| #F9FAFB | #1F2937 | 15.1:1 | ✅ 优秀 |

### 4.3 避免的配色组合

❌ **不推荐**：
- 浅灰色文字 + 浅色背景
- 红色文字 + 绿色背景（色盲问题）
- 相近颜色的搭配

✅ **替代方案**：
- 加深文字颜色
- 或加深背景颜色
- 或使用互补色

## 五、Emoji 和符号使用指南

### 5.1 使用原则

1. **适量使用**：每张图 3-5 个即可
2. **相关性**：emoji 必须与内容相关
3. **一致性**：同系列文章使用相同风格的 emoji
4. **可读性**：emoji 不能影响文字阅读

### 5.2 常用 Emoji 组合

#### 技术类
```
🚀 成长/加速
💡 创意/想法
⚡ 快速/高效
🎯 目标/精准
📊 数据/分析
🔧 工具/技术
💻 代码/编程
🤖 AI/智能
```

#### 成长类
```
🌱 成长/萌芽
📈 提升/进步
🏆 成就/获奖
⭐ 推荐/精选
✓ 完成/确认
🎉 庆祝/成功
```

#### 情感类
```
💖 喜欢/推荐
🔥 热门/火爆
✨ 精彩/亮点
💭 思考/感悟
❤️ 热爱/喜爱
```

### 5.3 使用位置

- ✅ 列表项前：`🚀 技巧1`
- ✅ 标题后：`AI编程 💻`
- ✅ 强调重点：`重要 ⚡`
- ❌ 不要在句子中滥用
- ❌ 不要使用过多导致花哨

## 六、整体设计检查清单

在设计公众号封面或配图时，使用以下清单检查：

### 文字排版
- [ ] 主标题是否醒目清晰？
- [ ] 副标题是否从属于主标题？
- [ ] 文字层级是否分明？
- [ ] 留白是否合理（不过于拥挤）？
- [ ] 是否支持小屏幕阅读？

### 视觉元素
- [ ] 背景是否简洁但不单调？
- [ ] 装饰元素是否服务于内容？
- [ ] 配色是否和谐统一？
- [ ] 是否有视觉吸引力？

### 对比度和可读性
- [ ] 文字与背景对比度是否足够（≥4.5:1）？
- [ ] 在不同设备上是否清晰可读？
- [ ] 色盲用户是否能够区分？

### 整体风格
- [ ] 是否符合小红书风格特征（清新、精致、有设计感）？
- [ ] 是否保持专业感？
- [ ] 是否适合年轻人审美？
- [ ] 同系列文章风格是否统一？

## 七、与其他风格系统的融合

### 7.1 与公众号特殊要求的融合

小红书风格需要与公众号的特殊要求结合：
- **比例适配**：3:4（小红书）→ 2.35:1（公众号封面）
- **安全区域**：关键内容必须在中心 60% 区域
- **双重场景**：同时考虑封面展示和分享卡片裁剪
- **横屏布局**：从竖屏思维转变为横屏设计

### 7.2 风格模板系统

小红书风格可以作为多种风格模板之一，与其他风格并存：
- 专业科技（保留公众号原有风格）
- 清新活泼（融入小红书风格）
- 简约极简（极简主义）
- 温暖治愈（情感化设计）
- 商务专业（商务风格）

## 八、实际应用建议

### 8.1 封面设计流程

1. **确定主题**：明确核心信息
2. **选择风格**：根据内容选择合适的风格模板
3. **设计标题**：主标题 + 副标题（如有）
4. **选择配色**：使用风格模板的配色方案
5. **添加装饰**：适量的图标或几何图形
6. **检查留白**：确保不拥挤
7. **测试对比度**：确保文字清晰可读
8. **验证安全区域**：确保关键内容居中

### 8.2 配图设计流程

1. **明确内容**：确定要传达的信息
2. **选择页面类型**：封面/内容/总结
3. **布局设计**：根据页面类型选择布局
4. **添加图标**：使用相关图标增强表达
5. **色彩统一**：与封面保持配色一致
6. **检查可读性**：确保信息清晰传达

### 8.3 风格一致性

系列文章的风格一致性建议：
- 使用相同的配色方案
- 使用相同的字体风格
- 使用相似的装饰元素
- 保持相同的整体氛围

## 九、常见问题

### Q1: 如何平衡小红书风格和公众号专业感？

**A**: 关键在于"度"的把握：
- 保留小红书的排版、留白、配色优点
- 适度使用 emoji（不要滥用）
- 选择适合专业场景的配色（如专业科技风）
- 保持内容的严肃性，只在视觉上增加吸引力

### Q2: 封面和配图风格如何统一？

**A**:
1. 使用同一风格模板（配色、字体、装饰）
2. 保持相同的视觉元素（如图标风格）
3. 调整比例：封面 2.35:1，配图 16:9
4. 封面更强调冲击力，配图更强调可读性

### Q3: 如何判断留白是否合理？

**A**:
- 视觉测试：是否有"呼吸感"？
- 小屏幕测试：手机上是否清晰？
- 信息密度测试：是否一眼能抓住重点？
- 对比测试：与同类设计对比，是否过于拥挤或空旷？
