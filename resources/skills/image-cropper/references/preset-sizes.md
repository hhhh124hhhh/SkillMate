## 尺寸计算

### 2.35:1（微信封面）

```
如果高度为383，宽度应为：383 × 2.35 ≈ 900
如果高度为460，宽度应为：460 × 2.35 ≈ 1080
```

### 16:9（标准横版）

```
如果高度为1080，宽度应为：1080 × 16/9 = 1920
如果高度为720，宽度应为：720 × 16/9 = 1280
```

### 1:1（正方形）

```
宽度 = 高度
例如：900x900、1024x1024、1080x1080
```

## 安全区域设计

### 微信公众号封面

由于分享时会自动裁剪为1:1，设计时遵循：

```
画面布局（2560x1440为例）：
[边缘30%] [中心40%安全区] [边缘30%]
  可裁剪      所有文字内容      可裁剪
```

**关键原则**：
- 所有关键内容（标题、副标题、核心图标）必须放在中心40-60%区域
- 边缘区域只能放装饰性元素
- 设计时先用1:1框测试，确保核心内容完整

**测试方法**：
1. 生成2.35:1封面（900x383）
2. 同时生成1:1测试版（900x900）
3. 检查1:1版本中所有文字是否完整
4. 调整设计直到两个版本都满意

## 常见问题

### Q：为什么微信封面推荐2.35:1而不是16:9？

A：微信公众号官方推荐2.35:1比例（900x383），这是最适合头条展示的尺寸。16:9也可以使用，但可能会有黑边。

### Q：如何确保分享后文字不被裁掉？

A：遵循安全区域原则，将所有关键内容放在画面中心60%区域。建议设计时使用辅助线标记出安全区域。

### Q：可以自定义尺寸吗？

A：可以！使用 `--size` 参数指定任意尺寸，格式为 `宽x高`（如 `1200x630`）。

### Q：批量裁剪时会保留原始文件吗？

A：会的。批量裁剪会创建新的输出目录，原始文件不受影响。
