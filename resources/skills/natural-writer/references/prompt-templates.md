## 模板1：全能去AI味（推荐）

**适用场景**：通用文案优化、公众号文章、自媒体内容

**使用方式**：复制以下模板，替换`{}`中的内容

```
请根据以下要求重写这篇文章：

【人设】你是一名{角色}，对{主题}持{支持/反对/怀疑/客观}态度

【风格】写作风格界于书面和口语之间，自然流畅

【技巧】
1. 使用高多样性和高节奏感
   - 词汇丰富多变
   - 长短句交替
   - 避免固定模式

2. 避免AI套路化表达
   - 不使用"首先、其次、最后"
   - 不使用"综上所述、总而言之"
   - 不使用"值得注意的是"、"显而易见"

3. 增强人味儿
   - 加入口语化词汇（"说实话"、"emm"、"啊对了"）
   - 加入个人观点和情感表达
   - 使用比喻和修辞手法
   - 增加细节描写和具体案例

4. 优化句子结构
   - 多用短句，每个句子表达一个意思
   - 避免长难句和复杂从句
   - 删除文末总结部分，自然收尾

【原文】
{原文内容}

【字数要求】
{字数}字左右
```

**示例**：
```
【人设】你是一名科技博主，对AI编程工具持支持态度
【风格】写作风格界于书面和口语之间，自然流畅
【原文】
AI编程工具可以帮助开发者提高效率。首先，AI可以自动生成代码。其次，AI可以检查错误。最后，AI可以优化性能。
【字数要求】
300字左右
```

## 模板2：情感文案专用

**适用场景**：情绪激发、情感共鸣、故事化内容

**使用方式**：复制以下模板，替换`{}`中的内容

```
写一篇关于{主题}的文案，字数{字数}。

【情感基调】{温暖/感人/激励/幽默/怀旧}

【场景设定】
{具体场景描述，包括时间、地点、环境}

【人物视角】
{第一人称/第三人称}

【写作要求】
1. 详细描写场景细节
   - 视觉：颜色、光线、形状
   - 听觉：声音、音乐、噪音
   - 嗅觉：气味、氛围

2. 加入人物对话
   - 使用口语化表达
   - 包含真实对话特征（打断、犹豫、笑场）
   - 展现人物性格和关系

3. 使用比喻和象征手法
   - 用自然现象隐喻（破茧、潮汐、四季更替）
   - 避免直接说出情绪名称
   - 让读者自己感受

4. 结合文化背景和生活情境
   - 时代特有的元素
   - 共同回忆和梗
   - 代际之间的差异和融合

5. 考虑读者可能的情感状态
   - 承认困难的存在（不回避痛苦）
   - 提供具体可行的微行动建议
   - 用真实案例而非空洞说教

6. 口语化表达，避免机械感
   - 多用短句和反问句
   - 加入情感词汇（"说实话"、"这让我很震撼"）
   - 删除所有逻辑词和总结

【目标读者】
{目标读者群体}
```

**示例**：
```
写一篇关于"年终总结"的文案，字数800字。

【情感基调】温暖而感人

【场景设定】
12月31日的晚上，一个人坐在咖啡馆里，窗外飘着小雪

【人物视角】
第一人称

【目标读者】
在城市打拼的年轻人
```

## 模板3：快速润色

**适用场景**：已生成内容的后期优化

**使用方式**：复制以下模板，替换`{}`中的内容

```
请修改下面的文章，使其更像人类写的：

【文章内容】
{原文}

【修改要求】
1. 口语化、自然化
2. 调整用词、修辞、逻辑结构
3. 减弱AI痕迹
4. 先指出不像人类的地方，再修改

【分析】
（AI会先分析这段文字中不够像人类回答的地方）

【修改后】
（AI给出修改版本）

【字数要求】
保持在{字数}字左右
```

**示例**：
```
【文章内容】
综上所述，AI技术对教育行业产生了深远影响。首先，AI可以个性化教学。其次，AI可以减轻教师负担。最后，AI可以提高学习效率。

【字数要求】
保持在200字左右
```

## 模板4：学术降重

**适用场景**：技术报告、数据分析、论文降重

**使用方式**：复制以下模板，替换`{}`中的内容

```
对以下论文/报告内容进行降AI处理：

【原文】
{论文/报告内容}

【处理步骤】
1. 简化句子结构
   - 将长句拆分为简洁、明了的短句
   - 每个句子聚焦于表达一个具体意思
   - 避免使用过于复杂的复合句或从句

2. 减少专业术语
   - 避免使用过多专业术语或难懂的术语
   - 尽量采用常见、通俗易懂的表达方式
   - 确保内容能够让广泛的读者群体理解

3. 替换过渡词
   - 将"首先、其次、最后"替换为最基础的词语
   - 删除"综上所述、总而言之"等总结性词汇
   - 使用更自然的过渡方式

4. 提高可读性
   - 使用简洁、易懂的语言
   - 确保文章结构清晰、段落内容简洁
   - 每一段落围绕一个清晰的主题展开

5. 保持学术严谨性
   - 不改变核心信息和数据
   - 保持专业度
   - 确保逻辑关系清晰

【注意事项】
- 保持学术严谨性的同时增加可读性
- 不要改变核心数据和结论
- 确保逻辑关系清晰
```

**示例**：
```
【原文】
本研究通过多种实验方法，深入探讨了AI模型在自然语言处理领域的应用。首先，我们采用了Transformer架构。其次，我们引入了预训练机制。最后，我们验证了模型的有效性。

【注意事项】
保持学术严谨性的同时增加可读性，不要改变核心数据和结论
```

## 模板5：口语化改写

**适用场景**：社交媒体、短视频脚本、直播文案

**使用方式**：复制以下模板，替换`{}`中的内容

```
请将以下内容改写为口语化版本：

【原文】
{原文内容}

【改写要求】
1. 加入口语化词汇
   - "说实话"、"emm"、"啊对了"
   - "这让我很震撼"
   - "你知道吗"、"举个例子"

2. 在关键词前后添加情感表达
   - 个人观点和偏好
   - 情感反应
   - 真实感受

3. 多运用短句，而不是长句
   - 每句控制在15字以内
   - 像说话一样自然

4. 穿插语气词
   - "呢"、"吧"、"哦"、"啊"
   - 但不要过度使用

5. 删除所有逻辑词
   - 不用"首先、其次、最后"
   - 不用"综上所述"

【目标】
让内容像是在和朋友聊天一样自然
```

## 模板6：高多样性改写

**适用场景**：避免AI检测、提升内容独特性

**使用方式**：复制以下模板，替换`{}`中的内容

```
请使用高多样性和高节奏感改写这篇文章：

【原文】
{原文内容}

【改写标准】

多样性标准：
- 词汇丰富度和不可预测性
- 避免重复使用相同的词语和句式
- 使用同义词替换
- 变换表达方式

节奏感标准：
- 句子长度和句式的波动幅度
- 长短句交替
- 避免固定模式
- 让内容更生动和富有层次感

【具体技巧】
1. 使用比喻和象征
2. 变换句子结构（陈述、疑问、感叹、倒装）
3. 调整段落长度
4. 增加修辞手法（排比、对偶、反问）
5. 加入个人化表达

【字数】
{字数}字左右
```

**态度示例**：
- 支持、反对、怀疑、客观

- 温暖、感人、激励、幽默、怀旧

### 3. 最佳实践

✅ **推荐做法**：
- 第一次生成时就使用这些模板
- 结合具体场景调整参数
- 人工审校后再次优化

❌ **避免做法**：
- 不要一次生成过长内容（建议1000字以下）

## 常见问题

### Q1: 如何判断去AI味效果？

A: 可以使用以下方法：
1. AI检测工具（如Copyscape、Turnitin）
2. 人工阅读评估（更自然、更像人类）
3. 读者反馈（是否有情感共鸣）

### Q2: 去AI味会影响内容质量吗？

A: 不会。正确的去AI味：
- 保留核心信息和数据
- 提升可读性和自然度
- 增强情感共鸣
- 让内容更有吸引力

### Q3: 哪些模型效果更好？

A: 根据实践，以下模型效果较好：
- Claude（自然度最高）
- DeepSeek（中文优化好）
- 豆包（情感表达好）

### Q4: 需要多次润色吗？

A: 建议：
1. 第一次生成时就使用去AI味模板
2. 如果还有AI味，使用"快速润色"模板
3. 人工审校后微调
